  * Cuadrícula de Calendario Mensual: Muestra una cuadrícula con los días del mes actual.
   * Navegación entre Meses: Puedes moverte al mes anterior o siguiente usando las teclas de flecha izquierda (left o h) y derecha (right
     o l).
   * Selección de Día: Puedes seleccionar un día específico dentro del mes usando las teclas de flecha arriba (up o k) y abajo (down o
     j).
   * Resaltado de Días:
       * El día actual (hoy) se resalta con un color diferente.
       * El día seleccionado por el usuario también se resalta, con un color distinto al de "hoy" para diferenciarlo.
   * Indicadores de Tareas y Eventos: Los días que tienen tareas o eventos asociados muestran un pequeño punto (•) junto al número del
     día.
   * Panel de Elementos del Día Seleccionado: En la parte inferior de la vista, se muestra un panel que lista todas las tareas y eventos
     programados para el día que tienes seleccionado. Si no hay elementos, indicará "No items for this day."
